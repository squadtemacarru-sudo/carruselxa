# Copy Frameworks (Hopkins / Kennedy / Schwartz)

**Core Principle**: Frameworks encode decades of tested persuasion sequences. Use them as skeletons, then fill with brand voice.

## AIDA — Attention > Interest > Desire > Action
Best for: slides, emails, ads. Most versatile framework.
```
[A] Hook que detiene el scroll
[I] "Acá está la clave..." — info que mantiene enganchado
[D] "Imaginá que..." — el after-state pintado con detalle
[A] CTA específico y claro
```

## PAS — Problem > Agitate > Solution
Best for: carruseles educativos, audiencias que sienten el dolor.
```
[P] El problema nombrado con las palabras EXACTAS que usa la audiencia
[A] Consecuencias reales. Lo que pasa si esto no se resuelve.
[S] La solución. Mecanismo claro. CTA.
```

## BAB — Before > After > Bridge
Best for: carruseles cortos, ads, posts de impacto.
```
[Antes] "Ahora mismo, probablemente..."
[Después] "Imaginá en cambio..."
[Puente] "[Marca/método] es cómo pasás de uno al otro."
```

## FAB — Feature > Advantage > Benefit
Best for: slides de producto, cuando hay que traducir características en valor.
```
[F] "Tiene X"
[A] "Lo que significa que Y"
[B] "Para que vos puedas Z"
```
Siempre terminá en el BENEFICIO EMOCIONAL, no en la característica técnica.

## 4Ps — Promise > Picture > Proof > Push
Best for: slides de conversión, CTA final de carrusel.
```
[Promesa] La afirmación central
[Imagen] El después pintado vívidamente
[Prueba] Testimonial, número, caso real
[Push] CTA con urgencia o razón para actuar hoy
```

## Framework Selection Guide
| Carrusel tipo | Framework recomendado |
|---|---|
| Educativo / "por qué" | PAS |
| Transformación / antes-después | BAB |
| Lanzamiento / oferta | 4Ps |
| General / versátil | AIDA |
| Producto / servicio específico | FAB |

## Para carruseles Instagram
- El slide 1 (cover) = [A] de AIDA / [P] de PAS
- Los slides intermedios = [I]+[D] / [A]+[S]
- El slide final (CTA) = siempre termina con un push claro
